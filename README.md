# インターバルタイマー

筋トレ用のインターバルタイマーアプリケーションです。カウントダウンとインターバルを設定して、効率的なトレーニングをサポートします。

## 特徴

- ⏱️ **カウントダウンタイマー**: トレーニング時間をカウントダウン
- 🔄 **インターバルタイマー**: 休憩時間を自動で管理
- 🎨 **シンプルなデザイン**: GoogleのWebアプリのようなクリーンなUI
- 🔊 **音声通知**: カウントダウン・インターバル終了時に音で通知
- 📱 **画面常時ON**: Wake Lock APIでタイマー中は画面が自動で切れない
- 🔗 **URL状態管理**: 設定をURLクエリパラメータで保存・共有可能
- 📊 **視覚的フィードバック**: 円形プログレスバーで残り時間を表示

## 使い方

1. **カウントダウンとインターバルの秒数を設定**
   - スライダーで0〜120秒の範囲で設定可能（デフォルト: 10秒）

2. **開始ボタンをクリック**
   - カウントダウンが開始されます
   - 画面が自動で常時ONになります

3. **自動でループ**
   - カウントダウン終了後、自動でインターバルに切り替わります
   - インターバル終了後、再びカウントダウンが開始されます

4. **停止・リセット**
   - 停止ボタンで一時停止
   - リセットボタンで初期状態に戻す

## 機能詳細

### タイマー設定

- **カウントダウン**: トレーニング時間（0〜120秒）
- **インターバル**: 休憩時間（0〜120秒）
- 設定値はURLクエリパラメータ（`?countdown=10&interval=10`）で保存されます

### 音声通知

- カウントダウン終了時: 800Hzのビープ音
- インターバル終了時: 600Hzのビープ音
- チェックボックスで音のON/OFFを切り替え可能

### 画面常時ON機能

- Wake Lock APIを使用して、タイマー実行中は画面が自動で切れないようにします
- モバイルデバイスでのトレーニング時に便利です
